Frequently Asked Questions

Here some common questions that may pop up as you use ProjSync:

Creating a project is the first step to getting work done in ProjSync! If you're a project manager, you have the ability to add new projects. To create a project, click the 'Create Project' button on the application's home page. A pop-up menu will open where we enter the project information. Here's what every new project needs:

Mandatory
  • Project name
  • Project key
  • Project type
  • Start date
  • Due date
Optional
  • Project description
  • Project budget

When you're done, click the 'Create' button to add your new project.
